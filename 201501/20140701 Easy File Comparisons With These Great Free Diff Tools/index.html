еҠ иҪҪзӣ®еҪ•дёӯ...

ж–Үд»¶иҪ»жқҫжҜ”еҜ№пјҢдјҹеӨ§иҖҢиҮӘз”ұзҡ„жҜ”иҫғиҪҜд»¶д»¬

ж–Үд»¶жҜ”иҫғе·Ҙе…·з”ЁдәҺжҜ”иҫғи®Ўз®—жңәдёҠзҡ„ж–Үд»¶зҡ„еҶ…е®№пјҢжүҫеҲ°д»–д»¬д№Ӣй—ҙзӣёеҗҢдёҺдёҚеҗҢд№ӢеӨ„гҖӮжҜ”иҫғзҡ„з»“жһңйҖҡеёёиў«з§°дёәdiffгҖӮ

diffеҗҢж—¶д№ҹжҳҜдёҖдёӘеҹәдәҺжҺ§еҲ¶еҸ°зҡ„гҖҒиғҪиҫ“еҮәдёӨдёӘж–Үд»¶д№Ӣй—ҙдёҚеҗҢд№ӢеӨ„зҡ„и‘—еҗҚзҡ„ж–Үд»¶жҜ”иҫғзЁӢеәҸзҡ„еҗҚеӯ—гҖӮdiffжҳҜдәҺдәҢеҚҒдё–зәӘ70е№ҙд»Јж—©жңҹпјҢеңЁUnixж“ҚдҪңзі»з»ҹдёҠиў«ејҖеҸ‘еҮәжқҘзҡ„гҖӮdiffе°ҶдјҡжҠҠдёӨдёӘж–Үд»¶д№Ӣй—ҙдёҚеҗҢд№ӢеӨ„зҡ„йғЁеҲҶиҝӣиЎҢиҫ“еҮәгҖӮ

LinuxжӢҘжңүеҫҲеӨҡдёҚй”ҷзҡ„GUIе·Ҙе…·пјҢиғҪдҪҝдҪ иғҪжё…жҘҡзҡ„зңӢеҲ°дёӨдёӘж–Үд»¶жҲ–еҗҢдёҖж–Үд»¶дёҚеҗҢзүҲжң¬д№Ӣй—ҙзҡ„дёҚеҗҢд№ӢеӨ„гҖӮиҝҷж¬ЎжҲ‘д»ҺиҮӘе·ұжңҖе–ңж¬ўзҡ„GUIжҜ”иҫғе·Ҙе…·дёӯйҖүеҮәдәҶдә”дёӘжҺЁиҚҗз»ҷеӨ§е®¶гҖӮйҷӨдәҶе…¶дёӯзҡ„дёҖдёӘпјҢе…¶д»–зҡ„йғҪжҳҜејҖжәҗзҡ„гҖӮ

иҝҷдәӣеә”з”ЁзЁӢеәҸеҸҜд»Ҙи®©дҪ жӣҙжё…жҘҡзҡ„зңӢеҲ°ж–Үд»¶жҲ–зӣ®еҪ•зҡ„е·®еҲ«пјҢиғҪеҗҲе№¶жңүе·®ејӮзҡ„ж–Үд»¶пјҢеҸҜд»Ҙи§ЈеҶіеҶІзӘҒе№¶е°Ҷе…¶иҫ“еҮәжҲҗдёҖдёӘж–°зҡ„ж–Үд»¶жҲ–иЎҘдёҒпјҢе…¶д№ҹз”ЁдәҺйӮЈдәӣйў„и§Ҳе’ҢеӨҮжіЁж–Үд»¶ж”№еҠЁзҡ„дә§е“ҒдёҠпјҲжҜ”еҰӮпјҢеңЁжәҗд»Јз ҒеҗҲе№¶еҲ°жәҗж–Үд»¶ж ‘д№ӢеүҚпјҢиҰҒе…ҲжҺҘеҸ—жәҗд»Јз Ғзҡ„ж”№еҸҳпјүгҖӮеӣ жӯӨе®ғд»¬жҳҜйқһеёёйҮҚиҰҒзҡ„иҪҜд»¶ејҖеҸ‘е·Ҙе…·гҖӮе®ғд»¬еҸҜд»Ҙеё®еҠ©ејҖеҸ‘дәәе‘ҳд»¬еҜ№ж–Үд»¶иҝӣиЎҢеӨ„зҗҶпјҢдёҚеҒңзҡ„жҠҠж–Үд»¶иҪ¬жқҘиҪ¬еҺ»гҖӮиҝҷдәӣжҜ”иҫғе·Ҙе…·дёҚд»…д»…иғҪз”ЁдәҺжҳҫзӨәжәҗд»Јз Ғж–Үд»¶дёӯзҡ„дёҚеҗҢд№ӢеӨ„пјӣд»–д»¬иҝҳйҖӮз”ЁдәҺеҫҲеӨҡз§Қзҡ„ж–Үжң¬ж–Үд»¶гҖӮеҸҜи§ҶеҢ–зҡ„зү№жҖ§дҪҝж–Үд»¶жҜ”иҫғеҸҳеҫ—е®№жҳ“гҖҒз®ҖеҚ•гҖӮ

###Meld

MeldжҳҜдёҖдёӘйҖӮз”ЁдәҺGnomeжЎҢйқўзҡ„гҖҒејҖжәҗзҡ„гҖҒеӣҫеҪўеҢ–зҡ„ж–Үд»¶е·®ејӮжҹҘзңӢе’ҢеҗҲе№¶зҡ„еә”з”ЁзЁӢеәҸгҖӮе®ғж”ҜжҢҒ2еҲ°3дёӘж–Үд»¶зҡ„еҗҢж—¶жҜ”иҫғгҖҒйҖ’еҪ’ејҸзҡ„зӣ®еҪ•жҜ”иҫғгҖҒеӨ„дәҺзүҲжң¬жҺ§еҲ¶(Bazaar, Codeville, CVS, Darcs, Fossil SCM, Git, Mercurial, Monotone, Subversion)д№ӢдёӢзҡ„зӣ®еҪ•жҜ”иҫғгҖӮиҝҳиғҪеӨҹжүӢеҠЁжҲ–иҮӘеҠЁеҗҲе№¶ж–Үд»¶е·®ејӮгҖӮ

Meldзҡ„йҮҚзӮ№еңЁдәҺеё®еҠ©ејҖеҸ‘дәәе‘ҳжҜ”иҫғе’ҢеҗҲе№¶еӨҡдёӘжәҗж–Үд»¶пјҢе№¶еңЁд»–д»¬жңҖе–ңж¬ўзҡ„зүҲжң¬жҺ§еҲ¶зі»з»ҹдёӢиғҪзӣҙи§Ӯзҡ„жөҸи§Ҳж”№еҠЁиҝҮзҡ„ең°ж–№гҖӮ

еҠҹиғҪеҢ…жӢ¬

  • еҺҹең°зј–иҫ‘ж–Үд»¶пјҢеҚіж—¶жӣҙж–°

  • иҝӣиЎҢдёӨеҲ°дёүдёӘж–Үд»¶зҡ„жҜ”иҫғеҸҠеҗҲе№¶

  • еңЁжҳҫзӨәзҡ„е·®ејӮе’ҢеҶІзӘҒд№Ӣй—ҙзҡ„еҜјиҲӘ

  • дҪҝз”ЁжҸ’е…ҘгҖҒж”№еҸҳе’ҢеҶІзӘҒиҝҷеҮ з§Қж Үи®°еҸҜи§ҶеҢ–еұ•зӨәжң¬ең°е’Ңе…ЁеұҖзҡ„е·®ејӮ

  • еҶ…зҪ®жӯЈеҲҷиЎЁиҫҫејҸж–Үжң¬иҝҮж»ӨеҷЁпјҢеҸҜд»ҘеҝҪз•ҘдёҚйҮҚиҰҒзҡ„е·®ејӮ

  • иҜӯжі•й«ҳдә®еәҰжҳҫзӨәпјҲдҪҝз”ЁеҸҜйҖүзҡ„gtksourceview)

  • е°ҶдёӨеҲ°дёүдёӘзӣ®еҪ•дёӯзҡ„ж–Үд»¶йҖҗдёӘиҝӣиЎҢжҜ”иҫғпјҢжҳҫзӨәж–°е»әпјҢзјәеӨұе’ҢжӣҝжҚўиҝҮзҡ„ж–Үд»¶

  • еҜ№д»»дҪ•жңүеҶІзӘҒжҲ–е·®ејӮзҡ„ж–Үд»¶зӣҙжҺҘжү“ејҖжҜ”иҫғз•Ңйқў

  • еҸҜд»ҘиҝҮж»Өж–Үд»¶жҲ–зӣ®еҪ•д»ҘйҒҝе…Қд»ҘеҝҪз•Ҙжҹҗдәӣе·®ејӮ

  • иў«ж”№еҠЁеҢәеҹҹзҡ„иҮӘеҠЁеҗҲе№¶жЁЎејҸдҪҝеҗҲе№¶жӣҙе®№жҳ“

  • д№ҹжңүдёҖдёӘз®ҖеҚ•зҡ„ж–Үд»¶з®ЎзҗҶ

  • ж”ҜжҢҒеӨҡз§ҚзүҲжң¬жҺ§еҲ¶зі»з»ҹпјҢеҢ…жӢ¬Git, Mercurial, Bazaar е’Ң SVN

  • еңЁжҸҗдәӨеүҚејҖеҗҜж–Үд»¶жҜ”иҫғжқҘжЈҖжҹҘж”№еҠЁзҡ„ең°ж–№е’ҢеҶ…е®№

  • жҹҘзңӢж–Үд»¶зүҲжң¬зҠ¶жҖҒ

  • иҝҳиғҪиҝӣиЎҢз®ҖеҚ•зҡ„зүҲжң¬жҺ§еҲ¶ж“ҚдҪңпјҲдҫӢеҰӮпјҢжҸҗдәӨгҖҒжӣҙж–°гҖҒж·»еҠ гҖҒз§»еҠЁжҲ–еҲ йҷӨж–Үд»¶пјү

  • з»§жүҝиҮӘеҗҢдёҖж–Үд»¶зҡ„дёӨдёӘж–Үд»¶иҝӣиЎҢиҮӘеҠЁеҗҲе№¶

  • ж ҮжіЁе№¶еңЁдёӯй—ҙзҡ„зӘ—ж јжҳҫзӨәжүҖжңүжңүеҶІзӘҒзҡ„еҸҳжӣҙзҡ„еҹәзЎҖзүҲжң¬

  • жҳҫзӨәе№¶еҗҲе№¶еҗҢдёҖж–Үд»¶зҡ„ж— е…ізҡ„зӢ¬з«Ӣдҝ®ж”№

  • й”Ғе®ҡеҸӘиҜ»жҖ§иҙЁзҡ„еҹәзЎҖж–Үд»¶д»ҘйҒҝе…ҚеҮәй”ҷ

  • еҸҜд»Ҙж•ҙеҗҲеҲ°е·Іжңүзҡ„е‘Ҫд»ӨиЎҢз•ҢйқўдёӯпјҢеҢ…жӢ¬gitmergetool

  • еӣҪйҷ…еҢ–ж”ҜжҢҒ

  • еҸҜи§ҶеҢ–дҪҝж–Үд»¶жҜ”иҫғжӣҙз®ҖеҚ•

  • зҪ‘еқҖ: meldmerge.org

  • ејҖеҸ‘дәәе‘ҳ: Kai Willadsen

  • иҜҒд№Ұ: GNU GPL v2

###DiffMerge

DiffMergeжҳҜдёҖдёӘеҸҜд»ҘеңЁLinuxгҖҒWindowsе’ҢOS XдёҠиҝҗиЎҢзҡ„пјҢеҸҜд»ҘеҸҜи§ҶеҢ–ж–Үд»¶зҡ„жҜ”иҫғе’ҢеҗҲе№¶зҡ„еә”з”ЁиҪҜд»¶гҖӮ

еҠҹиғҪеҢ…жӢ¬:

  • еӣҫеҪўеҢ–жҳҫзӨәдёӨдёӘж–Үд»¶д№Ӣй—ҙзҡ„е·®еҲ«гҖӮеҢ…жӢ¬жҸ’е…ҘиЎҢпјҢй«ҳдә®ж ҮжіЁд»ҘеҸҠеҜ№зј–иҫ‘зҡ„е…Ёйқўж”ҜжҢҒ

  • еӣҫеҪўеҢ–жҳҫзӨәдёүдёӘж–Үд»¶д№Ӣй—ҙзҡ„е·®еҲ«гҖӮпјҲе®үе…Ёзҡ„еүҚжҸҗдёӢпјүе…Ғи®ёиҮӘеҠЁеҗҲе№¶пјҢе№¶еҜ№жңҖз»Ҳж–Үд»¶еҸҜд»ҘйҡҸж„Ҹзј–иҫ‘

  • е№¶жҺ’жҳҫзӨәдёӨдёӘж–Үд»¶еӨ№зҡ„жҜ”иҫғпјҢжҳҫзӨәе“ӘдёҖдёӘж–Үд»¶еҸӘеӯҳеңЁдәҺе…¶дёӯдёҖдёӘж–Үд»¶еӨ№иҖҢдёҚеӯҳеңЁдәҺеҸҰеӨ–зҡ„дёҖдёӘж–Үд»¶еӨ№пјҢиҝҳиғҪдёҖеҜ№дёҖзҡ„е°Ҷе®Ңе…ЁзӣёеҗҢзҡ„гҖҒзӯүд»·зҡ„жҲ–дёҚеҗҢзҡ„ж–Үд»¶й…ҚеҜ№

  • и§„еҲҷи®ҫзҪ®е’ҢйҖүйЎ№и®©дҪ еҸҜд»ҘдёӘжҖ§еҢ–е®ғзҡ„еӨ–и§Ӯе’ҢиЎҢдёә

  • еҹәдәҺUnicodeпјҢеҸҜд»ҘеҜје…ҘеӨҡз§Қзј–з Ғзҡ„еӯ—з¬Ұ

  • и·Ёе№іеҸ°е·Ҙе…·

###xxdiff

xxdiffжҳҜдёӘејҖжәҗзҡ„еӣҫеҪўеҢ–зҡ„еҸҜиҝӣиЎҢж–Үд»¶гҖҒзӣ®еҪ•жҜ”иҫғеҸҠеҗҲе№¶зҡ„е·Ҙе…·гҖӮ

xxdiffеҸҜд»Ҙз”ЁдәҺжҳҫзӨәдёӨеҲ°дёүдёӘж–Үд»¶жҲ–дёӨдёӘзӣ®еҪ•зҡ„е·®еҲ«пјҢиҝҳиғҪдә§з”ҹдёҖдёӘеҗҲе№¶еҗҺзҡ„зүҲжң¬гҖӮиў«жҜ”иҫғзҡ„дёӨеҲ°дёүдёӘж–Үд»¶дјҡе№¶жҺ’жҳҫзӨәпјҢе№¶е°ҶжңүеҢәеҲ«зҡ„ж–Үеӯ—еҶ…е®№з”ЁдёҚеҗҢйўңиүІй«ҳдә®жҳҫзӨәд»ҘдҫҝдәҺиҜҶеҲ«гҖӮ

иҝҷдёӘзЁӢеәҸжҳҜдёӘйқһеёёйҮҚиҰҒзҡ„иҪҜд»¶ејҖеҸ‘е·Ҙе…·гҖӮд»–еҸҜд»ҘеӣҫеҪўеҢ–зҡ„жҳҫзӨәдёӨдёӘж–Үд»¶жҲ–зӣ®еҪ•д№Ӣй—ҙзҡ„е·®еҲ«пјҢеҗҲе№¶жңүе·®ејӮзҡ„ж–Үд»¶пјҢе…¶д№ҹз”ЁдәҺйӮЈдәӣйў„и§Ҳе’ҢеӨҮжіЁж–Үд»¶ж”№еҠЁзҡ„дә§е“ҒдёҠпјҲжҜ”еҰӮпјҢеңЁжәҗд»Јз ҒеҗҲе№¶еҲ°жәҗж–Үд»¶ж ‘д№ӢеүҚпјҢиҰҒе…ҲжҺҘеҸ—жәҗд»Јз Ғзҡ„ж”№еҸҳпјү

еҠҹиғҪеҢ…жӢ¬:

  • жҜ”иҫғдёӨеҲ°дёүдёӘж–Үд»¶пјҢжҲ–жҳҜдёӨдёӘзӣ®еҪ•пјҲжө…еұӮжҲ–йҖ’еҪ’пјү

  • жЁӘеҗ‘й«ҳдә®жҳҫзӨәе·®ејӮ

  • дәӨдә’ејҸзҡ„ж–Үд»¶еҗҲе№¶пјҢеҸҜи§ҶеҢ–зҡ„иҫ“еҮәе’Ңдҝқеӯҳ

  • еҸҜд»Ҙиҫ…еҠ©еҗҲе№¶зҡ„иҜ„и®ә/зӣ‘з®Ў

  • иҮӘеҠЁеҗҲе№¶ж–Үд»¶дёӯж—¶дёҚеҗҲе№¶ CVS еҶІзӘҒпјҢе№¶д»ҘдёӨдёӘж–Үд»¶жҳҫзӨәд»ҘдҫҝдәҺи§ЈеҶіеҶІзӘҒ

  • еҸҜд»Ҙз”Ёе…¶е®ғзҡ„жҜ”иҫғзЁӢеәҸи®Ўз®—е·®ејӮпјҡйҖӮз”ЁдәҺGNU diffгҖҒSGI diffе’ҢClearCaseзҡ„cleardiffпјҢд»ҘеҸҠжүҖжңүдёҺиҝҷдәӣзЁӢеәҸиҫ“еҮәзӣёдјјзҡ„ж–Үд»¶жҜ”иҫғзЁӢеәҸгҖӮ

  • еҸҜд»ҘдҪҝз”Ёиө„жәҗж–Үд»¶е®һзҺ°е®Ңе…Ёзҡ„дёӘжҖ§еҢ–и®ҫзҪ®

  • з”Ёиө·жқҘж„ҹи§үе’ҢRudy WortelжҲ–SGIзҡ„xdiffе·®дёҚеӨҡпјҢдёҺжЎҢйқўзі»з»ҹж— е…і

  • еҠҹиғҪе’Ңиҫ“еҮәеҸҜд»Ҙе’Ңи„ҡжң¬иҪ»жқҫйӣҶжҲҗ

  • зҪ‘еқҖ: furius.ca/xxdiff

  • ејҖеҸ‘дәәе‘ҳ: Martin Blais

  • иҜҒд№Ұ: GNU GPL

  • зүҲжң¬еҸ·: 4.0

###Diffuse

DiffuseжҳҜдёӘејҖжәҗзҡ„еӣҫеҪўеҢ–е·Ҙе…·пјҢеҸҜз”ЁдәҺеҗҲе№¶е’ҢжҜ”иҫғж–Үжң¬ж–Үд»¶гҖӮDiffuseиғҪеӨҹжҜ”иҫғд»»ж„Ҹж•°йҮҸзҡ„ж–Үд»¶пјҢе№¶жҺ’жҳҫзӨәпјҢе№¶жҸҗдҫӣжүӢеҠЁиЎҢеҢ№й…Қи°ғж•ҙпјҢиғҪзӣҙжҺҘзј–иҫ‘ж–Үд»¶гҖӮDiffuseиҝҳиғҪд»ҺbazaarгҖҒCVSгҖҒdarcs, git, mercurial, monotone, Subversionе’ҢGNU RCS еә“дёӯиҺ·еҸ–зүҲжң¬з”ЁдәҺжҜ”иҫғеҸҠеҗҲе№¶гҖӮ

еҠҹиғҪеҢ…жӢ¬:

  • жҜ”иҫғд»»ж„Ҹж•°йҮҸзҡ„ж–Үд»¶пјҢе№¶жҺ’жҳҫзӨәпјҲеӨҡж–№еҗҲе№¶пјү

  • иЎҢеҢ№й…ҚеҸҜд»Ҙиў«з”ЁжҲ·дәәе·Ҙзҹ«жӯЈ

  • зӣҙжҺҘзј–иҫ‘ж–Үд»¶

  • иҜӯжі•й«ҳдә®

  • ж”ҜжҢҒBazaar, CVS, Darcs, Git, Mercurial, Monotone, RCS, Subversionе’ҢSVK

  • ж”ҜжҢҒUnicode

  • еҸҜж— йҷҗж’Өй”Җ

###Kompare

KompareжҳҜдёӘејҖжәҗзҡ„GUIеүҚз«ҜзЁӢеәҸпјҢеҸҜд»ҘеҜ№дёҚеҗҢжәҗж–Үд»¶д№Ӣй—ҙе·®ејӮзҡ„еҸҜи§ҶеҢ–е’ҢеҗҲе№¶гҖӮKompareеҸҜд»ҘжҜ”иҫғж–Үд»¶жҲ–ж–Үд»¶еӨ№еҶ…е®№зҡ„е·®ејӮгҖӮKompareж”ҜжҢҒеҫҲеӨҡз§Қdiffж јејҸпјҢе№¶жҸҗдҫӣеҗ„з§ҚйҖүйЎ№жқҘи®ҫзҪ®жҳҫзӨәзҡ„дҝЎжҒҜзә§еҲ«гҖӮ

дёҚи®әдҪ жҳҜдёӘжғіжҜ”иҫғжәҗд»Јз Ғзҡ„ејҖеҸ‘дәәе‘ҳпјҢиҝҳжҳҜеҸӘжғіжҜ”иҫғдёҖдёӢз ”з©¶и®әж–ҮжүӢзЁҝдёҺжңҖз»Ҳж–ҮжЎЈзҡ„е·®ејӮпјҢKompareйғҪжҳҜдёӘжңүз”Ёзҡ„е·Ҙе…·гҖӮ
