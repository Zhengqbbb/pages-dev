еңЁ Linux дёҠд»Ҙж ‘зҠ¶жҹҘзңӢж–Үд»¶е’ҢиҝӣзЁӢ

д»Ӣз»ҚдёүдёӘ Linux е‘Ҫд»ӨпјҡpsгҖҒpstree е’Ң tree д»Ҙзұ»дјјж ‘зҡ„ж јејҸжҹҘзңӢж–Үд»¶е’ҢиҝӣзЁӢгҖӮ

Linux жҸҗдҫӣдәҶдёҖдәӣж–№дҫҝзҡ„е‘Ҫд»ӨпјҢз”ЁдәҺд»Ҙж ‘зҠ¶еҲҶж”ҜеҪўејҸжҹҘзңӢж–Үд»¶е’ҢиҝӣзЁӢпјҢд»ҺиҖҢжҳ“дәҺжҹҘзңӢе®ғд»¬д№Ӣй—ҙзҡ„е…ізі»гҖӮеңЁжң¬ж–ҮдёӯпјҢжҲ‘д»¬е°Ҷд»Ӣз»Қ psгҖҒpstree е’Ң tree е‘Ҫд»Өд»ҘеҸҠе®ғд»¬жҸҗдҫӣзҡ„дёҖдәӣйҖүйЎ№пјҢиҝҷдәӣйҖүйЎ№еҸҜеё®еҠ©дҪ е°ҶжіЁж„ҸеҠӣйӣҶдёӯеңЁиҰҒжҹҘзңӢзҡ„еҶ…е®№дёҠгҖӮ

ps

жҲ‘д»¬з”ЁжқҘеҲ—еҮәиҝӣзЁӢзҡ„ ps е‘Ҫд»ӨжңүдёҖдәӣжңүи¶Јзҡ„йҖүйЎ№пјҢдҪҶжҳҜеҫҲеӨҡдәәд»ҺжқҘжІЎжңүеҲ©з”ЁиҝҮгҖӮиҷҪз„¶еёёз”Ёзҡ„ ps -ef жҸҗдҫӣдәҶжӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢзҡ„е®Ңж•ҙеҲ—иЎЁпјҢдҪҶжҳҜ ps -ejH е‘Ҫд»ӨеўһеҠ дәҶдёҖдёӘдёҚй”ҷзҡ„ж•ҲжһңгҖӮе®ғзј©иҝӣдәҶзӣёе…ізҡ„иҝӣзЁӢд»ҘдҪҝиҝҷдәӣиҝӣзЁӢд№Ӣй—ҙзҡ„е…ізі»еңЁи§Ҷи§үдёҠжӣҙеҠ жё…жҷ°вҖ”вҖ”е°ұеғҸиҝҷдёӘзүҮж®өпјҡ

$ ps -ejH

еҸҜд»ҘзңӢеҲ°пјҢжӯЈеңЁиҝҗиЎҢзҡ„ ps иҝӣзЁӢжҳҜеңЁ bash дёӯиҝҗиЎҢзҡ„пјҢиҖҢ bash жҳҜеңЁ ssh дјҡиҜқдёӯиҝҗиЎҢзҡ„гҖӮ

-exjf йҖүйЎ№еӯ—з¬ҰдёІжҸҗдҫӣдәҶзұ»дјјзҡ„и§ҶеӣҫпјҢдҪҶжҳҜеёҰжңүдёҖдәӣе…¶е®ғз»ҶиҠӮе’Ңз¬ҰеҸ·д»ҘзӘҒеҮәжҳҫзӨәиҝӣзЁӢзҡ„еұӮж¬Ўз»“жһ„жҖ§иҙЁпјҡ

е‘Ҫд»ӨдёӯдҪҝз”Ёзҡ„иҝҷдәӣйҖүйЎ№иЎЁзӨәпјҡ

-e  йҖүжӢ©жүҖжңүиҝӣзЁӢ
-j  дҪҝз”Ёе·ҘдҪңж јејҸ
-f  жҸҗдҫӣе®Ңж•ҙж јејҸеҲ—иЎЁ
-H  еҲҶеұӮжҳҫзӨәиҝӣзЁӢпјҲеҰӮпјҢж ‘зҠ¶ж јејҸпјү
-x  еҸ–ж¶ҲвҖңеҝ…йЎ»дёҺ tty зӣёе…іиҒ”вҖқзҡ„йҷҗеҲ¶

еҗҢж—¶пјҢиҜҘе‘Ҫд»Өд№ҹжңүдёҖдёӘ --forest йҖүйЎ№жҸҗдҫӣдәҶзұ»дјјзҡ„и§ҶеӣҫгҖӮ

жіЁж„ҸпјҢиҝҷдәӣзӨәдҫӢеҸӘжҳҜиҝҷдәӣе‘Ҫд»ӨеҰӮдҪ•дҪҝз”Ёзҡ„зӨәдҫӢгҖӮдҪ еҸҜд»ҘйҖүжӢ©жңҖйҖӮеҗҲдҪ зҡ„иҝӣзЁӢи§Ҷеӣҫзҡ„д»»дҪ•йҖүйЎ№з»„еҗҲгҖӮ

pstree

tree
